# Kanban Board

Kanban-доска на vanilla JavaScript без фреймворков и библиотек.

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- Нативный HTML5 Drag & Drop API
- localStorage для хранения данных

## Функциональность

- Создание, редактирование и удаление карточек
- Перетаскивание карточек между колонками
- Фильтрация по приоритету и тегам
- Тёмная и светлая тема
- Адаптивный дизайн

## Запуск

Открыть `index.html` через Live Server в VS Code.
